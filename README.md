# Telegram Media Forwarder

Скрипт на Python с использованием библиотеки Telethon, который пересылает все существующие и новые медиафайлы (фото, видео, документы и т.д.) из одного Telegram-канала/чата в другой канал/чат или личный чат пользователя.

## Возможности

- Пересылка всех старых медиа из исходного чата при первом запуске.
- Мониторинг новых сообщений и автоматическая пересылка в реальном времени.
- Поддержка пересылки в личный чат пользователя (по username или номеру телефона).
- Игнорирование веб-страниц и опросов.
- Автоматическая обработка flood-wait от Telegram.

## Требования

- Python 3.7 или выше
- Установленный пакет `telethon`
- API ID и API Hash, полученные на [my.telegram.org/apps](https://my.telegram.org/apps)
